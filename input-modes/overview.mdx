GP2040-CE presents itself to the host as a different USB device depending on the selected input mode. Each mode uses a dedicated driver that handles report formatting, authentication (where required), and device-specific behavior.

## Supported modes

### Primary modes

| Enum value | Mode | Notes |
|---|---|---|
| `INPUT_MODE_XINPUT` (0) | XInput (Xbox 360/One) | No drivers needed on Windows |
| `INPUT_MODE_SWITCH` (1) | Nintendo Switch | Pro Controller emulation |
| `INPUT_MODE_PS3` (2) | PlayStation 3 | Simple HID, no auth required |
| `INPUT_MODE_KEYBOARD` (3) | Keyboard / Mouse HID | Fully remappable key mapping |
| `INPUT_MODE_PS4` (4) | PlayStation 4 | Auth required for console use |
| `INPUT_MODE_XBONE` (5) | Xbox One | Requires USB host peripheral |
| `INPUT_MODE_PS5` (13) | PlayStation 5 | Auth required for console use |
| `INPUT_MODE_GENERIC` (14) | Generic HID | Standard HID gamepad |
| `INPUT_MODE_SWITCH_PRO` (15) | Switch Pro Controller | Enhanced Switch compatibility |
| `INPUT_MODE_P5GENERAL` (16) | PS5 General (PC only) | Non-authenticated PS5 use on PC |
| `INPUT_MODE_XBOXORIGINAL` (12) | Original Xbox | Legacy Xbox support |

## Switching input modes

GP2040-CE has two mechanisms for selecting an input mode.

### Boot-time hotkey

Hold one of the mapped buttons while plugging in the controller. Each button (B1–B4, L1, L2, R1, R2) can be assigned a boot mode in the web configurator under **Settings → Boot Input Mode**. When you hold a button during power-on, the firmware starts in the mode assigned to that button.

<Steps>
  <Step title="Open the web configurator">
    Connect the controller to a PC and navigate to the GP2040-CE web configurator.
  </Step>
  <Step title="Go to Settings → Boot Input Mode">
    Each of the eight buttons (B1, B2, B3, B4, L1, L2, R1, R2) can be assigned an input mode.
  </Step>
  <Step title="Save and reboot">
    Save the settings. On next power-on, hold the corresponding button to start in that mode.
  </Step>
</Steps>

### Default mode

Set a persistent default mode in the web configurator under **Settings → Input Mode**. The controller will always start in this mode unless a boot-time button is held.

### Runtime hotkeys

Up to 16 hotkey combinations can be configured. Hotkeys are triggered by holding a combination of buttons while the controller is connected. The `HOTKEY_REBOOT_DEFAULT` action reboots into the default input mode.

<Info>
The firmware stores the active input mode across reboots. The next time the controller is plugged in without any boot buttons held, it resumes in the last-used default mode.
</Info>

## Authentication requirements

Some modes require additional hardware or credentials to operate on console hardware.

| Mode | PC | Console |
|---|---|---|
| XInput | No auth needed | N/A (PC only) |
| PS3 | No auth needed | No auth needed |
| PS4 | No auth needed | Requires auth (keys or USB dongle) |
| PS5 | No auth needed | Requires auth (USB dongle) |
| Xbox One | Requires USB host | Requires USB host + auth |

<Warning>
Without authentication, PS4 and PS5 modes work as a standard HID gamepad on PC but will not be recognized as a valid controller by PS4 or PS5 console hardware. Configure authentication before using these modes on a console.
</Warning>
