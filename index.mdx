GP2040-CE (Community Edition) is open-source gamepad firmware for the Raspberry Pi Pico and other RP2040-based boards. It delivers sub-millisecond input latency, supports 14+ input modes, and includes a built-in browser-based configurator — no software installation required.

Compatible with PC, PS3, PS4, PS5, Nintendo Switch, Xbox One, Steam Deck, MiSTer, and Android.

## Get started in minutes

<Steps>
  <Step title="Download the firmware">
    Grab the latest `.uf2` firmware file for your board from the [GP2040-CE releases page](https://github.com/OpenStickCommunity/GP2040-CE/releases).
  </Step>
  <Step title="Flash to your board">
    Hold the BOOTSEL button on your Pico while plugging it in via USB. It will appear as a USB drive — drag and drop the `.uf2` file onto it.
  </Step>
  <Step title="Open the web configurator">
    Unplug and replug your controller while holding the designated web configurator button (default: **S2**). Navigate to `http://192.168.7.1` in your browser.
  </Step>
  <Step title="Configure and play">
    Map your pins, select your input mode, enable add-ons, and save. Your controller is ready to use.
  </Step>
</Steps>

## Why GP2040-CE?

  <Card title="Ultra-Low Latency" icon="bolt">
    Average input latency of **0.76ms** in XInput mode at 1000Hz polling — competitive with top-tier commercial controllers.
  </Card>
  <Card title="14+ Input Modes" icon="plug">
    XInput, PS3, PS4, PS5, Nintendo Switch, Xbox One, Keyboard/HID, and more — switchable on the fly via hotkey.
  </Card>
  <Card title="Built-in Web UI" icon="globe">
    Configure everything through a browser. No drivers, no desktop app — just plug in and go.
  </Card>
  <Card title="Powerful Add-ons" icon="puzzle-piece">
    Turbo, analog input, RGB LEDs, OLED display, input macros, USB host passthrough, and many more.
  </Card>
  <Card title="SOCD Cleaning" icon="filter">
    Multiple SOCD modes including Up Priority, Neutral, Second Input Priority, First Input Priority, and Bypass.
  </Card>
  <Card title="Open Source" icon="code">
    MIT licensed. Build your own board configs, contribute add-ons, or fork the project for your needs.
