GP2040-CE ships with pre-built firmware for a wide range of RP2040-based boards. Each board has its own configuration in the `configs/` directory of the repository — the directory name corresponds to the firmware build name you need to download.

<Note>
  For the Raspberry Pi Pico, download the **Pico** build. For the Raspberry Pi
  Pico W, download the **PicoW** build. These are separate firmware files even
  though the hardware is nearly identical.
</Note>

## Breakout boards
